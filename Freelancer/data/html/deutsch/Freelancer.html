Bereiche der Softwareentwicklung

Desktopanwendungen

Dies ist alles, was man auf seinem Computer ausführt. Darunter fallen Beispielsweise Bildbearbeitungsprogramme, Programme für die Buchhaltung oder Simulationsprogramme. Meistens verwende ich dafür Java. Man kann es als Konsolenanwendung oder mit einer Grafischen Oberfläche gestalten, was allerdings mehr Zeit kostet.

Webentwicklung

Wenn man einen kleinen Laden hat oder aus irgend einem Grund von Leuten im Internet gefunden werden möchte, empfiehlt es sich eine Webseite zu haben. Diese könnte ich mit CSS schön gestalten und mit JavaScript einige Funktionalitäten oder Animationen mit einbringen. Es ist auch möglich Cookies zu setzen. Und da JavaScript immer im Browser des Anwenders ausgeführt wird, muss man sich auch keine Gedanken über den Rechenaufwand machen. Es ist auch möglich auf einer Webseite Formulare mit einzubinden und sie ausgefüllt an einen Server zu übermitteln. Dies nimmt allerdings mehr Zeit für die Erstellung in Anspruch.

Data Science

Data Science ist das maschinelle Einlesen großer Datenmengen meistens von Webseiten, um beispielsweise eine Statistik zu erstellen. Dazu ein Beispiel: Man möchte eine neue Immobilie kaufen und wissen, wo man für das wenigste Geld den meisten Wohnraum bekommt. Eine Suchmaschine wird da in den meisten fällen nicht weiterhelfen aber man kann ein Programm schreiben was aus dem Quellcode von Webseiten die Immobilienpreise isoliert, downloadet und in eine Datenbank schreibt, mit der man das beste Angebot errechnen kann.

Mikrocontroller / Embedded System

Fast jedes Gerät was Strom verbraucht hat einen Mikrocontroller. Hierzu wieder ein Beispiel: Wenn man eine einfache Kaffeemaschine bauen möchte, braucht man nur einen Taster zum starten, einen Stromausgang für das Mahlwerk und einen für die Wasserpumpe. Nun kann man die beiden Dinge nicht einfach mit dem Taster kurz schließen da sie eine bestimmte Zeit aktiv seien sollen. Dafür benutzt man einen Mikrocontroller. Er kann den Taster überwachen und die Komponenten eine gewünschte Zeitspanne lang mit Strom versorgen. Zusammengefasst könnte man sagen das ein Mikrocontroller ein sehr kleiner und günstiger Computer ist der sich programmieren lässt, aber nur wenige Signale einlesen, verarbeiten und ausgeben kann.

Datenbanken

Datenbanken muss man sich wie eine Ansammlung von Excel Tabellen vorstellen. Jedoch ist in einer Datenbank jede Tabelle mit anderen Tabellen verknüpft. Daher kann man mit ihnen aus großen Datenmengen gewünschte Informationen herausfiltern. Datenbanken kommen selten allein zum Einsatz. Man braucht auch immer eine Software die diese Verwaltet, um damit Beispielsweise seine Buchhaltung zu machen. Es ist auch möglich diese an Webseiten anzubinden.

Spielentwicklung

Hierbei ist zu beachten, dass Spiele bei denen man in einer virtuellen Welt umherlaufen kann, gerne mal 20 Leute für ein paar Jahre beschäftigen. Aufbauspiele hingegen sind für eine Person schaffbar. Man müsste dann noch einen Grafikdesigner engagieren der das Spiel gestaltet, da die Programmierung und das Gestalten von Bildern zwei unterschiedliche Aufgabenbereiche sind.
